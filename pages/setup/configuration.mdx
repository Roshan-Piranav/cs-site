# Configuring Default Settings

Define default behaviors to streamline your workflow.

## Available Configurations:
- *Default Export Format*: Choose between MDX, Markdown, or HTML.
- *Default Site*: Select the WordPress site to publish to.
- *Folder Mapping*: Automatically match Google Docs folders to WordPress categories.
