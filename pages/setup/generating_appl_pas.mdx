# Generating WordPress Application Password

Application passwords are needed to securely connect Docs with WordPress.

## Steps to Generate:

1. Go to your WordPress site → **Dashboard**  
2. Navigate to **Users → All Users → click your username**  
3. Scroll down to **Application Passwords**  
4. Enter a name (e.g., "Docs") and click **Add New Application Password**

# Connecting Docs to WordPress

5. Copy the password shown and enter it in Docs settings.

Once entered, Docs will be connected and ready to publish directly to WordPress.
