# Adding New WordPress Sites

You can manage multiple WordPress sites from a single Docs TM dashboard.

### 1. Once the application password is generated, you can now connect your WordPress site inside Google Docs.

### 2. Navigate to Google Docs

- Click on **Docs TM to WordPress Pro → Push to WordPress → Inside the Sidebar click on configure → Add new site**

### 4. Fill in the following details:

- **Site URL**: Your WordPress site (e.g., `https://yourdomain.com`)
- **Username**: Your WordPress admin username
- **Application Password**: Paste the password you generated earlier

> These will be in your **WordPress Dashboard → Users → All Users → Your Username**

### 5. Click **Add Site**

If the connection is successful, your site will appear in the **Connected Sites** list.  
You can now start exporting documents to WordPress directly from Google Docs.

> **Tip:** You can manage multiple WordPress sites using this extension by repeating the process for each.
