# Installation Guide

This guide helps you install the _Docs TM to WordPress Pro_ extension within Google Docs.

## 1. Prerequisites

- A Google account
- An active WordPress site
- Admin access to your WordPress site (for later steps)

## 2. Installing the Extension

To install the Docs TM to WordPress Pro extension:

1. Open any Google Doc.
2. Click on the top menu:  
   **Extensions → Add-ons → Get Add-ons → Search for Docs TM to WordPress Pro → Download and Install**

3. The extension will be added automatically to your Google Docs.

**You should now see _Docs TM to WordPress Pro_ listed under the Extensions menu.**

> Once installed, you can start exporting your docs directly to WordPress from within Google Docs.
