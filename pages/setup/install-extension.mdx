# Installation Guide

This guide helps you install the _Docs to WordPress Pro_ extension within Google Docs.

## 1. Prerequisites

- A Google account
- An active WordPress site
- Admin access to your WordPress site (for later steps)

## 2. Installing the Extension

To install the Docs to WordPress Pro extension:

1. Open any Google Doc.
2. Click on the top menu:  
   **Extensions → Add-ons → Get Add-ons → Search for "Docs to WordPress Pro" → Install**

Or install it directly from the [Google Workspace Marketplace](https://workspace.google.com/marketplace/app/docs_to_wordpress_pro/1074215048146).
