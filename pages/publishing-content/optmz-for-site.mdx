## Site-Specific Settings

Each site can have its own:

- Default categories
- Post type
- Author
- SEO settings (if enabled)
- Featured image rules

## To Configure

1. Go to **Settings > Site Preferences** in the extension.
2. Choose a site.
3. Adjust defaults and SEO templates.

> Pro Tip: Use custom fields for integration with SEO plugins like Yoast or RankMath.
