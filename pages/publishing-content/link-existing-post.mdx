# Linking to an Existing Post

Link a Google Doc to an existing WordPress post to make edits easier.

## Steps

1. Open the Google Doc.
2. In the extension, click **Link to Existing Post**.
3. Search by post title or ID.
4. Select the post to link.
5. Once linked, any updates from this doc can sync directly.

## Why Link?

- Keeps your workflow inside Google Docs.
- Ensures consistent formatting and version control.
